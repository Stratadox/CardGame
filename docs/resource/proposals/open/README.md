# Open Proposals
The open proposals of a [player account](../../account/overview/README.md) 
are the [match proposals](../overview/README.md) they can currently accept.

- Open proposals have not been accepted nor expired yet.

## Embedded
- A list of [match proposals](../overview/README.md)
